Soy economista apasionado por el análisis de datos, la automatización, el desarrollo web y la divulgación económica. Me dedico a transformar números en historias con impacto, combinando métodos cuantitativos con herramientas modernas de programación y diseño digital.

## Qué hago

- **Investigación económica**: análisis de inflación, política monetaria, finanzas públicas y estudios de mercado personalizados para emprendedores.
- **Desarrollo web**: creo landing pages, sitios corporativos y plataformas de captación de leads con diseño profesional, SEO optimizado y versiones bilingües (EN/ES).
- **Ciencia de datos**: modelos predictivos, visualización interactiva, dashboards y automatización de reportes con Python.
- **Gestión de bases de datos**: limpieza, deduplicación, consultas SQL, migraciones y preparación de datos para campañas de email marketing.

## Tecnologías que domino

**Backend y datos:** Python · Pandas · SQL · PostgreSQL · Google Apps Script · BeautifulSoup · Selenium  
**Frontend y web:** HTML5 · CSS3 · JavaScript · Dash · Plotly  
**Herramientas:** Git · GitHub Pages · Google Analytics · Power BI · LaTeX · Jinja2

## Formación

- Licenciatura en Economía
- Diplomado en Data Science y Machine Learning
- Desarrollo web y SEO aplicado
